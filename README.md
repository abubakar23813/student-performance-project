 Student Performance Prediction

This is my first beginner Data Science project where I explored how students' reading and writing scores affect their math performance.

---

🔹 About the Project

I created this project as part of my learning journey in Data Science. The goal was to understand basic data analysis and build a simple machine learning model.

---

🔹 What I Did

- Cleaned the dataset by removing missing values
- Created new features like total and average score
- Visualized the data using graphs
- Built a Linear Regression model to predict math scores

---

🔹 Tools & Technologies

- Python
- Pandas
- Matplotlib
- Scikit-learn

---

🔹 Model Result

The model achieved an accuracy of around 0.97 (R² score) on the dataset.

Since this is a small dataset, the result may not reflect real-world performance.

---

🔹 What I Learned

- Basics of data cleaning
- Data visualization techniques
- How machine learning models work
- Importance of train-test split

---

🔹 How to Run

pip install pandas, matplotlib, scikit-learn
python main.py

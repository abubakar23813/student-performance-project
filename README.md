Student Performance Prediction
=====================================

This is my first Data Science project, and I have implemented it using Python.

🔹 Overview of the Project
---------------------------

In this project, I have worked on the analysis of the performance of students, and the prediction of the marks that students are likely to score in mathematics.

🔹 Features
------------

- Data Cleaning using Pandas
- Feature Engineering
- Data Visualization
- Machine Learning Model
- Linear Regression Algorithm

🔹 Technologies Used
-------------------

- Python
- Pandas
- Matplotlib
- Scikit-learn

🔹 Output
------------

It predicts the marks that students are likely to score in mathematics on the basis of the marks that they have scored in reading and writing.
